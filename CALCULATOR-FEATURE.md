# 🧮 ฟีเจอร์เครื่องคำนวณตัวอย่าง - ทันสมัย!

## ✨ **ฟีเจอร์ใหม่:**

### 📊 **เครื่องคำนวณแบบเรียลไทม์**
ช่อง "ระยะเวลา (วัน)" ไม่ได้ใช้สำหรับยืมจริง แต่เป็น **เครื่องมือทดลองคำนวณ** ว่า:

**"ถ้ายืมแล้วคืนช้า จะต้องจ่ายเท่าไหร่?"**

---

## 🎯 **วิธีใช้งาน:**

### 1. กรอกจำนวนเงิน
```
เช่น: 100 บาท
```

### 2. คลิกปุ่มตัวอย่างด่วน หรือกรอกเอง
```
[1 วัน]  [3 วัน]  [6 วัน]  [10 วัน]
```

### 3. ดูผลลัพธ์แบบเรียลไทม์
```
ยอดที่ต้องชำระ: 105.00 บาท
ดอกเบี้ย: 5% (5.00 บาท)
```

---

## 📱 **ตัวอย่างการใช้งาน:**

### สถานการณ์ที่ 1: ต้องการยืม 100 บาท
```
1. กรอก: 100 บาท
2. คลิกปุ่ม "1 วัน" 
   → ยอดชำระ: 100.00 บาท (ไม่มีดอก) ✅

3. คลิกปุ่ม "6 วัน" 
   → ยอดชำระ: 105.00 บาท (ดอก 5%) ⚠️
   
4. คลิกปุ่ม "10 วัน"
   → ยอดชำระ: 109.00 บาท (ดอก 9%) 🔴
```

**สรุป:** "โอ้! ถ้าคืนช้า 10 วัน ต้องจ่ายเพิ่ม 9 บาท"

### สถานการณ์ที่ 2: เช็คค่าดอกเบี้ยก่อนยืม
```
ผู้ใช้: "ถ้าผมยืม 50 บาท แล้วคืนวันที่ 5 ต้องจ่ายเท่าไหร่?"

1. กรอก: 50 บาท
2. คลิกปุ่ม "6 วัน" (หรือกรอก 5)
   → ยอดชำระ: 52.00 บาท (ดอก 2%)

ผู้ใช้: "โอเค เข้าใจแล้ว! ถ้าช้า 4 วัน เพิ่มแค่ 2 บาท"
```

---

## 🎨 **UI/UX ที่ปรับปรุง:**

### 1. ปุ่มด่วน 4 ปุ่ม
```
✅ [1 วัน]   - สีเขียว (ไม่มีดอก)
⚠️ [3 วัน]   - สีเหลือง (ดอกเล็กน้อย)
⚠️ [6 วัน]   - สีชมพู (ดอกปานกลาง)
🔴 [10 วัน]  - สีแดง (ดอกสูง)
```

### 2. คำอธิบายชัดเจน
```
💡 "คลิกปุ่มด้านบนเพื่อดูตัวอย่างการคำนวณแบบเรียลไทม์"

ℹ️ หมายเหตุ:
• ระบบจะยืมให้ 1 วัน เท่านั้น
• ช่อง "ระยะเวลา" ใช้สำหรับ ทดลองคำนวณ
• เช่น: คลิกปุ่ม "6 วัน" เพื่อดูว่าถ้าคืนในวันที่ 6 ต้องจ่ายเท่าไหร่
```

### 3. การแสดงผลแบบเรียลไทม์
```
เปลี่ยนวัน → ผลลัพธ์เปลี่ยนทันที
1 วัน → สีเขียว (ไม่มีดอก)
6 วัน → สีแดง (มีดอก 5%)
```

## 💻 **ตัวอย่าง Code:**

### HTML - ปุ่มด่วน
```html
<button onclick="setDays(1)">1 วัน</button>
<button onclick="setDays(3)">3 วัน</button>
<button onclick="setDays(6)">6 วัน</button>
<button onclick="setDays(10)">10 วัน</button>
```

### JavaScript - ฟังก์ชันเปลี่ยนวัน
```javascript
function setDays(days) {
    document.getElementById('borrowDays').value = days;
    calcBorrowSummary(); // คำนวณทันที
}
```

## ✅ **ประโยชน์:**

1. **โปร่งใส** - ผู้ใช้รู้ล่วงหน้าว่าถ้าคืนช้าต้องจ่ายเท่าไหร่
2. **ทันสมัย** - คำนวณแบบเรียลไทม์ไม่ต้องรอ
3. **ง่าย** - คลิกปุ่มเดียว ได้คำตอบทันที
4. **ชัดเจน** - มีคำอธิบายว่าช่องนี้ใช้ทำอะไร
