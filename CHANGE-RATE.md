## 🧮 **สูตรคำนวณ:**

```
ดอกเบี้ย/วัน = ยอดยืม × 1%
ดอกเบี้ยรวม = ดอกเบี้ย/วัน × (จำนวนวัน - 1)
ยอดชำระ = ยอดยืม + ดอกเบี้ยรวม
```

### ตัวอย่าง: ยืม 100 บาท 5 วัน
```
ดอกเบี้ย/วัน = 100 × 0.01 = 1.00 บาท
ดอกเบี้ยรวม = 1.00 × (5-1) = 4.00 บาท
ยอดชำระ = 100 + 4.00 = 104.00 บาท
```
