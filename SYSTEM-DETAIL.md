# 📘 ระบบ MONEY - คู่มือรายละเอียดฉบับสมบูรณ์

## 🎯 กฎการยืมเงิน

### ⏰ **ระยะเวลา**
- ยืมได้สูงสุด: **1 วัน (24 ชั่วโมง)**
- นับเวลาตั้งแต่: วินาทีที่แอดมินกดอนุมัติ
- ครบกำหนด: 24 ชั่วโมงหลังอนุมัติ

### 💰 **การชำระเงิน**

#### ✅ **คืนภายใน 24 ชั่วโมง**
```
ยอดยืม = ยอดคืน (ไม่มีดอกเบี้ย)
ตัวอย่าง:
- ยืม 10 บาท → คืน 10 บาท
- ยืม 50 บาท → คืน 50 บาท  
- ยืม 100 บาท → คืน 100 บาท
```

#### ⚠️ **เกิน 24 ชั่วโมง (วันที่ 2)**
```
ยอดคืน = ยอดยืม + ดอกเบี้ย 2%
+ หัก 1 เครดิต

ตัวอย่าง:
- ยืม 10 บาท → คืน 10.20 บาท (ดอก +0.20)
- ยืม 50 บาท → คืน 51.00 บาท (ดอก +1.00)
- ยืม 100 บาท → คืน 102.00 บาท (ดอก +2.00)
- ยืม 500 บาท → คืน 510.00 บาท (ดอก +10.00)
- ยืม 1000 บาท → คืน 1020.00 บาท (ดอก +20.00)

เครดิตเหลือ: 60 - 1 = 59 เครดิต
```

#### ⚠️ **เกิน 48 ชั่วโมง (วันที่ 3)**
```
ยอดคืน = ยอดยืม + ดอกเบี้ย 4% (2% x 2 วัน)
+ หัก 2 เครดิต

ตัวอย่าง:
- ยืม 10 บาท → คืน 10.40 บาท (ดอก +0.40)
- ยืม 100 บาท → คืน 104.00 บาท (ดอก +4.00)
- ยืม 1000 บาท → คืน 1040.00 บาท (ดอก +40.00)

เครดิตเหลือ: 60 - 2 = 58 เครดิต
```

#### 🔴 **เกิน 7 วัน**
```
ยอดคืน = ยอดยืม + ดอกเบี้ย 12% (2% x 6 วัน)
+ หัก 6 เครดิต

ตัวอย่าง:
- ยืม 100 บาท → คืน 112.00 บาท (ดอก +12.00)
- ยืม 1000 บาท → คืน 1120.00 บาท (ดอก +120.00)

เครดิตเหลือ: 60 - 6 = 54 เครดิต
```

## 🎯 ระบบเครดิต

### เครดิตเริ่มต้น
```
สมัครใหม่ = 60 เครดิต ฟรี!
```

### การหักเครดิต
```
เกินกำหนด 1 วัน = -1 เครดิต
เกินกำหนด 2 วัน = -2 เครดิต
เกินกำหนด 3 วัน = -3 เครดิต
...
เกินกำหนด 60 วัน = -60 เครดิต (หมดเครดิต!)
```

### เมื่อเครดิตหมด (0)
```
❌ ไม่สามารถยืมเงินได้อีก
❌ ต้องคืนเงินที่ค้างทั้งหมดก่อน
```
