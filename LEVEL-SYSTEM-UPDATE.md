# 🎮 ระบบเลเวลและแต้ม - อัพเดตใหม่!

## ✨ **สิ่งที่เปลี่ยนแปลง:**

### 1. 📋 **ฟอร์มยืมเงิน: เพิ่มช่องกรอก**
- ✅ **เบอร์วอเลท** (เช่น 0812345678)
- ✅ **ชื่อบัญชี** (เช่น สมชาย ใจดี)
- ✅ ต้องกรอกก่อนกดยืมเงิน

### 2. 🎯 **ระบบแต้มและเลเวลใหม่**
- ❌ เปลี่ยนจาก "60 เครดิต" → ✅ **50 แต้ม (Points)**
- ✅ แต้มเริ่มต้น: **50 แต้ม, เลเวล 1**
- ✅ คืนตรงเวลา (ภายใน 24 ชม.) = **+1 แต้ม**
- ✅ คืนช้า = **-1 แต้มต่อวัน**

### 3. 📊 **ระบบเลเวล 1-20**
- เลเวลขึ้นตามแต้มที่สะสม
- ยิ่งเลเวลสูง ยืมได้มากขึ้น

## 🎯 **วิธีการได้แต้ม:**

### ✅ **ได้แต้ม +1:**
```
1. ยืมเงิน
2. แอดมินอนุมัติ → เวลา 24 ชม. เริ่มนับถอยหลัง
3. คืนเงินภายใน 24 ชม.
4. แอดมินอนุมัติสลิป
5. ระบบให้แต้ม +1 อัตโนมัติ 🎉
```

### ⚠️ **หักแต้ม -1 ต่อวัน:**
```
1. ยืมเงิน
2. แอดมินอนุมัติ → เวลา 24 ชม. เริ่มนับถอยหลัง
3. เกิน 24 ชม. แล้วยังไม่คืน
4. ระบบหักแต้ม 1 แต้ม/วัน อัตโนมัติ
5. เช่น: คืนช้า 5 วัน = หัก 4 แต้ม (วันแรกไม่นับ)
```

## 💻 **Backend Changes:**

### server.js - Registration:
```javascript
const newUser = {
    userId: 'user' + Date.now(),
    username,
    password,
    phone,
    wallet,
    role: 'user',
    points: 50,  // เริ่มต้น 50 แต้ม
    level: 1,    // เลเวลเริ่มต้น
    maxDailyBorrow: 100,  // เลเวล 1 ยืมได้ 100฿
    completedBorrows: 0   // จำนวนครั้งที่คืนตรงเวลา
};
```

### server.js - Calculate Level:
```javascript
function calculateLevel(points) {
    if (points >= 1000) return 20;
    if (points >= 950) return 19;
    if (points >= 900) return 18;
    // ... ต่อเนื่องไปจนถึง
    if (points >= 100) return 2;
    return 1;
}

user.level = calculateLevel(user.points);
user.maxDailyBorrow = 50 + (user.level * 50);
```

### server.js - Approve Slip (คืนเงิน):
```javascript
const now = new Date();
const dueDate = new Date(borrow.dueDate);
const returnedOnTime = now <= dueDate;

borrow.returnedOnTime = returnedOnTime;

// If returned on time, give +1 point
if (returnedOnTime) {
    user.points += 1;  // +1 แต้ม
    user.level = calculateLevel(user.points);
    user.maxDailyBorrow = 50 + (user.level * 50);
    
    // Check level up
    if (user.level > oldLevel) {
        io.emit('level-up', {
            username: user.username,
            newLevel: user.level,
            newLimit: user.maxDailyBorrow
        });
    }
}
```
