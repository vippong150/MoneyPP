# 🎉 อัพเดตระบบ MONEY - ระบบยืมเงินออนไลน์พรีเมี่ยม

## ✨ ฟีเจอร์ใหม่ที่เพิ่มเข้ามา

### 1. 📱 ระบบสมัครสมาชิกแบบใหม่
- ✅ ต้องกรอก **เบอร์โทรศัพท์** และ **เบอร์วอเลท** (ตรวจสอบรูปแบบ 0812345678)
- ✅ เบอร์โทรแต่ละเบอร์ใช้ได้แค่ครั้งเดียว (ป้องกันสมัครซ้ำ)
- ✅ บัญชีใหม่ได้รับ **60 เครดิต** ฟรีทันที

### 2. 💰 ระบบยืมเงิน - เงื่อนไขใหม่
- ✅ บัญชีใหม่ยืมได้ **1-100 บาท/วัน**
- ✅ **คืนภายใน 24 ชั่วโมง** = ไม่มีดอกเบี้ย คืนเท่าเดิม
- ✅ **เกิน 24 ชั่วโมง** = คิดดอกเบี้ย **10%** ของยอดเงินที่ยืม
  - ยืม 10 บาท → ต้องชำระ 11 บาท  
  - ยืม 100 บาท → ดอกเบี้ย 1 บาท/วันที่เกิน

### 3. 🎯 ระบบเครดิต
- ✅ บัญชีใหม่มี **60 เครดิต**
- ✅ ไม่คืนเงินตามกำหนด → **หัก 1 เครดิตต่อวัน**
- ✅ เครดิตหมด (0) → **ไม่สามารถยืมได้**
- ✅ ระบบตรวจสอบเครดิตอัตโนมัติ

### 4. 📸 ระบบอัปโหลดสลิปการชำระเงิน
- ✅ ผู้ใช้อัปโหลดหลักฐานการโอนเงิน (รูปภาพหรือ PDF)
- ✅ สลิปถูกส่งไปยัง **คลังแอดมิน** ทันที
- ✅ แอดมินตรวจสอบและอนุมัติผ่าน Admin Panel
- ✅ รองรับไฟล์: JPG, PNG, GIF, PDF (ขนาดไม่เกิน 5MB)

### 5. 🛡️ Admin Panel (หน้าใหม่)
- ✅ เข้าถึงได้ที่: `/admin.html`
- ✅ ตรวจสอบสลิปที่รออนุมัติ
- ✅ อนุมัติ/ปฏิเสธสลิปได้ทันที
- ✅ แสดงรูปภาพสลิปขนาดใหญ่ได้
- ✅ อัพเดตอัตโนมัติทุก 30 วินาที

### 6. 🎨 ดีไซน์พรีเมี่ยม
- ✅ UI/UX สวยงาม ทันสมัย
- ✅ รองรับ **มือถือ 100%** (Responsive)
- ✅ แอนิเมชันลื่นไหล
- ✅ สีสัน Gradient สไตล์พรีเมี่ยม

## 📂 ไฟล์ที่ถูกแก้ไข/เพิ่ม

### Backend (server.js)

- ✅ เพิ่มระบบ **Multer** สำหรับอัปโหลดไฟล์
- ✅ API `/api/auth/register` - รองรับเบอร์โทร + เบอร์วอเลท
- ✅ API `/api/user/info` - แสดงเครดิตและตรวจสอบรายการค้าง
- ✅ API `/api/borrow/request` - ตรวจสอบเครดิต + วงเงิน
- ✅ API `/api/borrow/upload-slip` - อัปโหลดสลิป
- ✅ API `/api/admin/slips` - ดูสลิปทั้งหมด (Admin)
- ✅ API `/api/admin/approve-slip` - อนุมัติสลิป (Admin)
- ✅ เพิ่มโฟลเดอร์ `/uploads` สำหรับเก็บสลิป

### Frontend
- ✅ **login.html** - เพิ่มช่องกรอกเบอร์โทร + เบอร์วอเลท + ตรวจสอบรูปแบบ
- ✅ **dashboard.html** - (ไฟล์เดิมสำรองเป็น dashboard.backup.html)
- ✅ **admin.html** - หน้าใหม่สำหรับ Admin ตรวจสอบสลิป

## 🚀 วิธีใช้งาน

1. เข้า http://localhost:8080

```json
  "creditDeducted": false

```

## 🎯 Status Codes
- `pending` - รอแอดมินอนุมัติคำขอยืม
- `approved` - แอดมินอนุมัติแล้ว รอผู้ใช้โอนเงิน
- `slip_uploaded` - ผู้ใช้อัปโหลดสลิปแล้ว รอแอดมินตรวจสอบ
- `completed` - เสร็จสิ้น แอดมินอนุมัติสลิปแล้ว
- `rejected` - ปฏิเสธ

## 🔐 การจัดการสิทธิ์
- **user** - ผู้ใช้ทั่วไป (ยืมเงิน, อัปโหลดสลิป)
- **admin** - ผู้ดูแลระบบ (อนุมัติคำขอ, ตรวจสอบสลิป)
- **dev** - นักพัฒนา (สิทธิ์เต็ม)
