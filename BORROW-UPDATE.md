# 🎯 อัพเดตฟอร์มยืมเงิน - ลบช่องระยะเวลา

## ✨ **สิ่งที่เปลี่ยนแปลง:**

### ❌ **ลบออก:**
1. ช่องกรอก "ระยะเวลา (วัน)" ในฟอร์มยืมเงิน
2. ช่องแสดง "ยอดที่ต้องชำระ" (แสดงแบบเรียลไทม์)
3. แถบแสดงดอกเบี้ยและยอดชำระด้านบน

### ✅ **เพิ่มใหม่:**
1. **เครื่องคำนวณตัวอย่าง** แบบใหม่:
   - มีปุ่มด่วน: `1 วัน` `3 วัน` `6 วัน` `10 วัน`
   - แสดงผลในกล่องแยกต่างหาก
   - ไม่กระทบกับฟอร์มยืมจริง

2. **ข้อความชัดเจน** เกี่ยวกับเวลา:
   - ยืมได้ 1 วัน (24 ชั่วโมง) เท่านั้น
   - เวลานับถอยหลังเริ่มเมื่อแอดมินอนุมัติ
   - คืนภายใน 24 ชม. = ไม่มีดอกเบี้ย
   - คืนช้า = คิดดอก 1%/วัน + หักแต้ม 1/วัน

## 🔄 **ขั้นตอนการยืมเงิน:**

### 1. ผู้ใช้ส่งคำขอ
```
• เลือกจำนวนเงิน (เช่น 100 บาท)
• กดส่งคำขอยืม
• รอแอดมินอนุมัติ
```

### 2. แอดมิน (tast2) อนุมัติ
```
• เข้าหน้า /admin.html
• กดปุ่ม "อนุมัติ"
• ⏱️ เวลานับถอยหลัง 24 ชั่วโมง เริ่มนับ!
```

### 3. ผู้ใช้คืนเงิน
```
สถานการณ์ที่ 1: คืนภายใน 24 ชั่วโมง
├─ จ่าย: 100 บาท
├─ ดอกเบี้ย: 0 บาท
└─ แต้มคงเหลือ: ไม่เปลี่ยนแปลง ✅

สถานการณ์ที่ 2: คืนช้า 3 วัน (72 ชั่วโมง)
├─ จ่าย: 102 บาท
├─ ดอกเบี้ย: 2 บาท (2%)
└─ แต้มถูกหัก: 2 แต้ม (วันละ 1) ⚠️

สถานการณ์ที่ 3: คืนช้า 10 วัน
├─ จ่าย: 109 บาท
├─ ดอกเบี้ย: 9 บาท (9%)
└─ แต้มถูกหัก: 9 แต้ม ❌
```

## 🧮 **เครื่องคำนวณตัวอย่าง:**

### วิธีใช้:
1. กรอกจำนวนเงิน (หรือเลือกจากแพ็กเกจ)
2. คลิกปุ่มตัวอย่าง: `1 วัน` / `3 วัน` / `6 วัน` / `10 วัน`
3. ดูผลลัพธ์ในกล่องด้านล่าง

### ตัวอย่าง:

#### ยืม 100 บาท
```
[✓ 1 วัน]  → ชำระ 100.00 บาท (ไม่มีดอก)
[⚠️ 3 วัน]  → ชำระ 102.00 บาท (ดอก 2%)
[⚠️ 6 วัน]  → ชำระ 105.00 บาท (ดอก 5%)
[❌ 10 วัน] → ชำระ 109.00 บาท (ดอก 9%)
```

#### ยืม 50 บาท
```
[✓ 1 วัน]  → ชำระ 50.00 บาท (ไม่มีดอก)
[⚠️ 3 วัน]  → ชำระ 51.00 บาท (ดอก 2%)
[⚠️ 6 วัน]  → ชำระ 52.50 บาท (ดอก 5%)
[❌ 10 วัน] → ชำระ 54.50 บาท (ดอก 9%)
```

### ✅ ฟังก์ชันใหม่:
```javascript
function previewDays(days) {
    // คำนวณตัวอย่างเท่านั้น
    const amount = parseFloat(document.getElementById('borrowAmount').value) || 100;
    
    let interestAmount = 0;
    let total = amount;
    
    if (days > 1) {
        const daysOverdue = days - 1;
        interestAmount = amount * 0.01 * daysOverdue; // 1% per day
        total = amount + interestAmount;
    }
    
    // แสดงผลในกล่อง calcPreview
    document.getElementById('previewAmount').textContent = amount + ' บาท';
    document.getElementById('previewDays').textContent = days + ' วัน';
    document.getElementById('previewInterest').textContent = interestAmount.toFixed(2) + ' บาท';
    document.getElementById('previewTotal').textContent = total.toFixed(2) + ' บาท';
    document.getElementById('calcPreview').style.display = 'block';
}
```

## 🎨 **UI/UX ที่ดีขึ้น:**

### ก่อน (เก่า):
```
❌ มีช่อง "ระยะเวลา (วัน)" ที่ทำให้สับสน
❌ ผู้ใช้คิดว่ายืมได้หลายวัน
❌ มีการคำนวณแบบเรียลไทม์ในฟอร์ม
❌ ไม่ชัดเจนว่าเวลานับจากตอนไหน
```

### หลัง (ใหม่):
```
✅ ไม่มีช่อง "ระยะเวลา" ในฟอร์ม
✅ ชัดเจนว่ายืมได้แค่ 1 วัน (24 ชั่วโมง)
✅ เครื่องคำนวณแยกออกมาชัดเจน
✅ บอกชัดว่าเวลานับเมื่อแอดมินอนุมัติ
✅ มีปุ่มตัวอย่างง่ายๆ คลิกเดียว
```

## ⚡ **ข้อดี:**

1. **ไม่สับสน** - ผู้ใช้ไม่ต้องกรอกจำนวนวัน
2. **ชัดเจน** - ยืมได้แค่ 1 วัน เท่านั้น
3. **เข้าใจง่าย** - เวลานับเมื่อแอดมินอนุมัติ
4. **มีเครื่องคำนวณ** - ดูตัวอย่างได้ง่าย
5. **ทันสมัย** - ปุ่มสีสันสวยงาม มีไอคอน
