# 📊 อัพเดตตาราง 1-31 วัน + คอลัมน์แต้มถูกหัก

## ✨ **การอัพเดตล่าสุด:**

### 🔥 **เพิ่มใหม่:**
1. **ตารางแสดง 1-31 วัน** (เพิ่มจาก 15 วัน)
2. **คอลัมน์ "แต้มถูกหัก"** แสดงว่าคืนช้ากี่วัน แต้มจะถูกหัก
3. **Scrollable Table** เลื่อนดูได้สูงสุด 500px

## 🎯 **กฎการหักแต้ม:**

### วันที่ 1 (คืนตรงเวลา)
```

✅ ดอกเบี้ย: 0%
✅ ยอดชำระ: 100฿ (ถ้ายืม 100฿)
```

### วันที่ 2-31 (คืนช้า)
```
⚠️ แต้มถูกหัก: -1 ถึง -30 แต้ม
⚠️ ดอกเบี้ย: 1-30%
⚠️ ยอดชำระ: 101-130฿ (ถ้ายืม 100฿)

สูตร:
• วันที่ N = หักแต้ม (N - 1) แต้ม
• วันที่ 5 = หักแต้ม 4 แต้ม
• วันที่ 15 = หักแต้ม 14 แต้ม
• วันที่ 31 = หักแต้ม 30 แต้ม
```

## 🎨 **ฟีเจอร์ใหม่:**

### 1. ⬆️ Sticky Header (หัวตารางติด)
```css
thead {
    position: sticky;
    top: 0;
    background: rgba(10,10,26,0.95);
    z-index: 1;
}
```
**ประโยชน์:** เลื่อนดูแถวล่างๆ ได้โดยหัวตารางยังเห็นอยู่

### 2. 📜 Max Height + Scroll
```css
overflow-y: auto;
max-height: 500px;
```
**ประโยชน์:** ตารางไม่สูงเกินไป มีแถบเลื่อนดูได้

### 3. 📱 Responsive Horizontal Scroll
```css
overflow-x: auto;
```
**ประโยชน์:** บนมือถือเลื่อนซ้าย-ขวาได้ ไม่พัง

### 4. 🎨 สีของคอลัมน์แต้ม
```javascript
color: pointsLost > 0 ? '#ff4757' : '#2ecc71'

วันที่ 1: สีเขียว (0 แต้ม)
วันที่ 2+: สีแดง (-1 ถึง -30)
```

## 💻 **JavaScript Code:**

```javascript
function generateCalcTable() {
    const amount = parseFloat(document.getElementById('borrowAmount').value) || 100;
    const tbody = document.getElementById('calcTableBody');
    tbody.innerHTML = '';
    
    // สร้าง 31 แถว
    for (let day = 1; day <= 31; day++) {
        let interestAmount = 0;
        let interestPercent = 0;
        let total = amount;
        let pointsLost = 0; // ใหม่!
        
        if (day === 1) {
            // วันที่ 1: ไม่มีดอก ไม่หักแต้ม
            pointsLost = 0;
            status = '✓ ไม่มีดอก';
            statusColor = '#2ecc71';
        } else {
            // วันที่ 2-31: มีดอก + หักแต้ม
            const daysOverdue = day - 1;
            interestAmount = amount * 0.01 * daysOverdue; // 1% per day
            interestPercent = 1.0 * daysOverdue;
            total = amount + interestAmount;
            pointsLost = daysOverdue; // หักแต้มเท่ากับจำนวนวันที่เกิน!
            
            // กำหนดสถานะและสี
            if (day <= 3) {
                status = '⚠️ ดอกต่ำ';
                statusColor = '#ffc107';
            } else if (day <= 7) {
                status = '⚠️ ดอกปานกลาง';
                statusColor = '#ff6b9d';
            } else {
                status = '❌ ดอกสูง';
                statusColor = '#ff4757';
            }
        }
        
        // สร้างแถว
        row.innerHTML = `
            <td>${day}</td>
            <td>${interestPercent.toFixed(0)}% (${interestAmount.toFixed(2)}฿)</td>
            <td>${total.toFixed(2)} ฿</td>
            <td style="color:${pointsLost > 0 ? '#ff4757' : '#2ecc71'};">
                ${pointsLost > 0 ? '-' + pointsLost : '0'}
            </td>
            <td>${status}</td>
        `;
    }
}
```

## ⚡ **ข้อดี:**

1. **ครบถ้วน** - เห็นข้อมูลครบ 31 วัน
2. **โปร่งใส** - รู้ว่าแต้มจะถูกหักกี่แต้ม
3. **เตือนภัย** - เห็นชัดว่าคืนช้าแพงแค่ไหน
4. **ใช้ง่าย** - เลื่อนดูได้ หัวตารางติด
5. **สวยงาม** - สีสัน hover effect
6. **Realtime** - เปลี่ยนจำนวนเงิน ตารางอัพเดตทันที

## 🚀 **วิธีเทส:**

1. เปิด `http://localhost:3000/dashboard.html`
2. ไปเมนู "ยืมเงิน"
3. กรอกจำนวนเงิน 100 บาท
4. คลิก **[แสดงตารางคำนวณ 1-31 วัน]**
5. ดูตารางเต็ม 31 แถว
6. เลื่อนลงดู - หัวตารางยังเห็นอยู่ (sticky!)
7. ดูคอลัมน์ "แต้มถูกหัก" - แสดง 0, -1, -2, ..., -30
8. เปลี่ยนเงินเป็น 50 หรือ 200 → ตารางอัพเดตทันที!
