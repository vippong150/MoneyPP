# 🐛 แก้ไขบัค - การคำนวณดอกเบี้ย

## ❌ **ปัญหาเดิม:**
```
ยืม 100 บาท
- กด 2 วัน → ยอดชำระ 110 บาท (ผิด!)
- กด 5 วัน → ยอดชำระ 110 บาท (ผิด!)

เหตุผล: ระบบคำนวณดอกเบี้ย 10% แบบคงที่
```

## ✅ **แก้ไขแล้ว:**
```
ยืม 100 บาท
- กด 1 วัน → ยอดชำระ 100.00 บาท (ไม่มีดอก) ✅
- กด 2 วัน → ยอดชำระ 100.20 บาท (ดอก 0.20 บาท) ✅
- กด 3 วัน → ยอดชำระ 100.40 บาท (ดอก 0.40 บาท) ✅
- กด 5 วัน → ยอดชำระ 100.80 บาท (ดอก 0.80 บาท) ✅
- กด 10 วัน → ยอดชำระ 101.80 บาท (ดอก 1.80 บาท) ✅
```

---

## 🔧 **การแก้ไข:**

### 1. เปลี่ยน CONFIG
```javascript
// เดิม (ผิด)
OVERDUE_INTEREST: 10  // 10%

// ใหม่ (ถูก)
OVERDUE_INTEREST_RATE: 0.002  // 0.20%
```

### 2. แก้ฟังก์ชันคำนวณ
```javascript
// เดิม (ผิด)
const interest = days > 1 ? CONFIG.OVERDUE_INTEREST : CONFIG.DEFAULT_INTEREST;
const total = amount + (amount * interest / 100);

// ใหม่ (ถูก)
if (days > 1) {
    const daysOverdue = days - 1;
    interestAmount = amount * CONFIG.OVERDUE_INTEREST_RATE * daysOverdue;
    interestPercent = 0.20 * daysOverdue;
    total = amount + interestAmount;
}
```

### 3. แก้การแสดงผล
```javascript
// แสดงทั้งเปอร์เซ็นต์และจำนวนบาท
document.getElementById('rateValue').textContent = 
    interestPercent.toFixed(2) + '% (' + interestAmount.toFixed(2) + ' บาท)';
```

## 🎯 **สูตรการคำนวณ:**

```
ดอกเบี้ย/วัน = ยอดยืม × 0.20%
ดอกเบี้ยรวม = ดอกเบี้ย/วัน × (จำนวนวัน - 1)
ยอดชำระ = ยอดยืม + ดอกเบี้ยรวม
```

### ตัวอย่าง: ยืม 100 บาท 5 วัน
```
ดอกเบี้ย/วัน = 100 × 0.002 = 0.20 บาท
ดอกเบี้ยรวม = 0.20 × (5-1) = 0.80 บาท
ยอดชำระ = 100 + 0.80 = 100.80 บาท
```
