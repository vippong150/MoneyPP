### 3. 📊 ระบบคำนวณดอกเบี้ยแบบเรียลไทม์
**สูตรคำนวณ: 2% ต่อวันที่เกิน (หรือ 10 บาท = 0.20 บาท)**

| ยอดยืม | ดอกเบี้ย/วัน (2%) | ยอดคืน (ถ้าเกิน 1 วัน) |
|--------|-------------------|------------------------|
| 10 บาท | 0.20 บาท | 10.20 บาท |
| 50 บาท | 1.00 บาท | 51.00 บาท |
| 100 บาท | 2.00 บาท | 102.00 บาท |
| 500 บาท | 10.00 บาท | 510.00 บาท |
| 1000 บาท | 20.00 บาท | 1020.00 บาท |

**หมายเหตุ:**
- คืนภายใน 24 ชม. = **ไม่มีดอกเบี้ย**
- เกิน 24 ชม. = **คิดดอกเบี้ย 2% ต่อวัน**

### 4. 🎯 Admin Panel แบบเต็มรูปแบบ
**URL: http://localhost:8080/admin.html**

#### Tab 1: คำขอรออนุมัติ
- ✅ แสดงคำขอยืมทั้งหมดที่รอ tast2 อนุมัติ

## 📋 ขั้นตอนการใช้งาน

### 🧑 สำหรับผู้ยืม:
1. เข้า http://localhost:8080
2. สมัครสมาชิก (กรอกเบอร์โทร + เบอร์วอเลท)
3. ล็อกอิน
4. กดยืมเงิน → กรอกจำนวน + **เลขบัญชีวอเลทที่จะรับเงิน**
5. รอ Admin อนุมัติ (tast2)
6. เมื่ออนุมัติแล้ว = รับเงินเข้าวอเลท
7. โอนคืน
8. อัปโหลดสลิป
9. รอ Admin อนุมัติสลิป = เสร็จสิ้น!

### 👨‍💼 สำหรับ Admin (tast2):
1. เข้า http://localhost:8080
2. ล็อกอินด้วย **tast2** / **1**
3. เข้า http://localhost:8080/admin.html
4. เห็น **Tab 1: คำขอรออนุมัติ**
   - เห็นข้อมูลผู้ยืมพร้อม**เลขวอเลท**
   - เห็นการคำนวณดอกเบี้ยแบบชัดเจน
   - คลิก "อนุมัติ" = โอนเงินไปให้ผู้ยืม
5. เมื่อผู้ยืมโอนคืน → อัปโหลดสลิป
6. เห็นใน **Tab 2: สลิปรอตรวจสอบ**
7. ดูรูปสลิป → คลิก "อนุมัติ" = เสร็จสิ้น!

## 🎯 Status Code
- `pending` = รอ tast2 อนุมัติ
- `approved` = tast2 อนุมัติแล้ว ผู้ยืมรับเงิน
- `slip_uploaded` = ผู้ยืมโอนคืน + อัปโหลดสลิป
