## 🐛 หากพบปัญหา

### ปัญหา: กดปฏิเสธแล้วยัง error
**แก้ไข:**
1. เปิด Browser Console (F12)
2. ดู error message
3. ตรวจสอบว่า server.js มี route `/api/admin/reject-borrow` หรือไม่
4. ตรวจสอบว่า `adminUsername` ถูกส่งไปใน request body หรือไม่

### ปัญหา: หน้า Dashboard ไม่แสดงนับถอยหลัง
**แก้ไข:**
1. เปิด Browser Console (F12)
2. ตรวจสอบว่ามี error ใน `loadActiveBorrows()` หรือไม่
3. ตรวจสอบว่า API `/api/borrow/history` ส่งข้อมูล `approved` มาหรือไม่
4. ตรวจสอบว่า `setInterval` ทำงานหรือไม่ (ดูใน Console)

### ปัญหา: Auto-Refresh ไม่ทำงาน
**แก้ไข:**
1. ตรวจสอบว่า `setInterval` อยู่ในโค้ด (บรรทัดสุดท้ายของ script)
2. เปิด Browser Console → เห็น API calls ทุก 5/15 วินาทีหรือไม่
3. ลองรีสตาร์ท server
